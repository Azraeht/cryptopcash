cryptopcash
===========

cryptopcash is a cli cryptocurrency portifolio


Installation
------------

.. code:: bash

    sudo pip install cryptopcash


Usage
-----


.. code:: bash
    cryptopcash



Colors:
------

Available colors :

    * black
    * yellow
    * white
    * dark red
    * light red
    * dark green
    * light green
    * brown
    * dark blue
    * light blue
    * dark magenta
    * light magenta
    * dark cyan
    * light cyan
    * dark gray
    * light gray
